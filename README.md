# IssueTracker

A full-stack issue tracking application built with Express GraphQL backend and React frontend.

## Project Structure

- **`backend/`** - GraphQL server using Express and GraphQL
- **`frontend/`** - React application for issue management

## Getting Started

### Backend Setup

```bash
cd backend
npm install
npm start
```

The API server runs on `http://localhost:3001` with GraphiQL explorer.

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

The React app runs on `http://localhost:3000`.

## Features

- Create, read, update, and manage issues
- Filter and search functionality
- GraphQL API for flexible data querying
- Real-time issue updates

## Technologies

- **Backend**: Node.js, Express, GraphQL
- **Frontend**: React, JavaScript, CSS
- **Database**: In-memory (can be extended with MongoDB, PostgreSQL, etc.)
